What is Synk?

Synk allows your applications to remember vital state using a simple web-based service. Synk-enabled applications can send and receive events — like “article 6 was flagged”, or “open a browser tab to http://rssynk.appspot.com” — so that you don't have to remember and repeat your actions.

Some applications are already using Synk, like the popular open source RSS/Atom newsreader Vienna. Synk developers are working with authors of other applications to help add support.

Synk is open source software, distributed under a permissive BSD-like license. Anyone is welcome to use synk for their own applications, and is encouraged to contribute changes back to the core Synk code base. Synk source code is avaialable at GitHub.

How do I get it?

Registering for Synk is a snap — it takes a total of about 3 seconds. Just choose a username and password, then enter it in your Synk-enabled applications on each of your computers, and the rest works like magic.
